siun

Prioritize package updates that matter.

siun helps you decide how important available package updates really are by calculating an urgency score. It supports various built-in criteria and allows users to implement their own in Python.

Features

  • Calculates urgency for package updates
  • Configurable criteria and thresholds
  • Supports Arch Linux (pacman) out of the box
  • Custom criteria via Python
  • Optional desktop notifications and news feeds

Installation

pip install siun

Usage

siun check

See the project page for documentation, advanced configuration guides, troubleshooting tips, and updates.
